Class dbObjProperty

Description
Description | Vars (details) | Methods (details)

A wrapper for linking objects in "single" relationships.

This describes the relationship between two objects in a ONE-TO-ONE relationship. It is the equivalent of the dbList class and is used in basically the same way.
